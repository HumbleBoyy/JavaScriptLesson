JavaScript Fundamentals

Data Types:

There are two data types in JavaScript Primitive and Non-Primitive
JavaScriptda ikki xil data turlari mavjud Primitive va Non-Primitive

Primitive:

Primitive data types are immutable, meaning their values cannot be changed once assigned.Storage: Primitive data types are stored directly in memory
Primitiv ma'lumotlar turlari o'zgarmasdir, ya'ni ularning qiymatlarini tayinlangandan keyin o'zgartirib bo'lmaydi.Saqlash: Primitiv ma'lumotlar turlari to'g'ridan-to'g'ri xotirada saqlanadi

Non-Primitive:

Non-primitive data types are mutable and can be modified.Whereas non-primitive data types are stored as references to their values in memory.
Primitiv bo'lmagan ma'lumotlar turlari o'zgaruvchan va o'zgartirilishi mumkin.Non-Primitive bo'lmagan ma'lumotlar turlari xotirada ularning qiymatlariga havola sifatida saqlanadi
